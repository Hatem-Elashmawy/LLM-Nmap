## How to set up a personal network using Virtual Box?
1) Install virutal box
2) Download VMs :
    - a virtual machine to execute requests through llm-nmap : *kali linux/ubuntu*
    - a "classic" target VM : *ubuntu server* (to open specifique port)
    - a "vurlnerable" target VM : *bee box* VM (a VM with many ports already open)
